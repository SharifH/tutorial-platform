ABOUT

Coding tutorials by example

Rundown is a platform to share code with instructions. There are many platforms offering tutorials to learn a programming language, and many to share code with others. Rundown attempts to provide a platform with step by step intructions for building a specific example (i.e. how to create a bouncing ball) or a larger project (i.e. how to create a website). It is a platform for creative coders at all levels that also gives you the flexibility to save your code, share it and even change mode from lesson mode to experimentation mode in order to test your own code and add to the examples on your own personal editor. Users can also access a template for creating their own project-based tutorial slideshow and share it on the website.
